Titanic Survival Prediction – Kaggle Competition Submission

Posted on 2025-09-2025

Tags: Python, Pandas, Scikit-Learn, ML Classification, Data Cleaning, Feature Engineering

Problem:

This project was a submission to the classic Titanic machine learning competition hosted on Kaggle. The task was to predict which passengers survived the Titanic shipwreck based on features such as age, sex, ticket class, and family relationships.

Approach:

  • Performed extensive data cleaning and preprocessing:
    • Handled missing values in the age and cabin features
    • Encoded categorical variables using label and one-hot encoding
    • Engineered new features such as family size and title extraction from names
  • Trained and evaluated several classification models including:
    • Logistic Regression
    • Random Forest
    • Gradient Boosting
  • Used cross-validation to assess model generalization
  • Submitted final predictions to Kaggle for leaderboard scoring

Outcome:

  • Achieved strong accuracy relative to the public leaderboard baseline
  • Demonstrated end-to-end machine learning workflow including:
    • Data wrangling
    • Feature engineering
    • Model selection
    • Deployment of submission-ready predictions
  • Used as a foundational ML project to solidify classification and data preprocessing concepts
