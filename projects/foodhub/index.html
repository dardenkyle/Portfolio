FoodHub Demand Analysis

Posted on 2025-09-2025

Tags: Python, Scikit-Learn, Pandas, ML Classification, Feature Engineering

Problem:

FoodHub, a fictional food delivery platform, wanted to identify customers likely to stop using their service so they could target retention campaigns more effectively. The challenge was to build a classification model that predicts churn with strong precision while minimizing false positives.

Approach:

  • Cleaned and explored customer behavioral data, including order history, frequency, and feedback
  • Engineered key features such as days since last order, satisfaction score, and delivery frequency
  • Trained multiple classification models including Logistic Regression, Decision Tree, and Random Forest
  • Optimized model using cross-validation and focused on improving precision to reduce unnecessary outreach
  • Identified top features influencing churn using feature importance scores

Outcome:

  • Achieved 98% precision on validation set using a tuned Random Forest model
  • Enabled the business to focus on high-probability churn customers for targeted retention efforts
  • Demonstrated potential cost savings by reducing broad incentive campaigns
